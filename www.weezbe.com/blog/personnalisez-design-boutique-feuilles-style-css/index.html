Personnalisez votre Design avec les feuilles de style (CSS)

Dernière mise à jour le 22/02/2017

Vous souhaitez faire évoluer votre design et le personnaliser à l’extrême ? Nous avons ouvert les accès aux feuilles de style (CSS). Pour cela, il suffit de vous rendre dans la rubrique « Configuration > CSS » ! Vous allez pouvoir modeler votre boutique comme vous le souhaitez.

Soyez vigilant car l’édition du CSS peut perturber le fonctionnement de votre boutique si vous ne respectez pas les standards du langage. Nous vous conseillons de ne pas y toucher si vous ne maîtrisez pas les rudiments du HTML et du CSS.

Application simple et concrète d’une modification de la feuille de style CSS

Voici un exemple simple de ce qui devient possible avec la personnalisation avancée. Vous pouvez mettre les liens de votre menu en gras avec le code si dessous !

#top #catalogue #menu > li > a {
	font-weight: bold;
}

Jusqu’où puis-je aller ?

Potentiellement, il n’y a aucune limite, vous pouvez redéfinir tous les styles CSS de votre boutique. Nous vous déconseillons néanmoins de modifier la grille CSS (structuration des pages) car la grille par défaut est responsive et garantie notamment un affichage correct sur tous les supports (smartphones, tablettes…).

À propos

Weezbe est un outil permettant de créer facilement et gratuitement une boutique en ligne professionnelle.
Adoptant un modèle économique à la performance, la solution e-commerce permet de lancer son activité sans coût de départ tout en profitant de fonctionnalités puissantes et étudiées pour la vente en ligne.
