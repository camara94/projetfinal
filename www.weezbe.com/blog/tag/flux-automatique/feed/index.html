Nous sommes ravis de vous annoncer que le comparateur de prix leDénicheur rejoint la liste des comparateurs de prix intégrés nativement à weezbe !

leDénicheur est un comparateur de prix et de produits dont l’objectif est d’aider les internautes à prendre les meilleures décisions d’achats. En plus d’un large catalogue de produits, celui-ci propose également de comparer les marchands entre-eux afin de trouver le meilleur rapport qualité / prix.

Afin de vous aider à apparaître dans ce comparateur, vous disposez désormais d’un flux produit disponible depuis la rubrique « Marketing > Comparateurs & Flux » de votre interface d’administration weezbe.

Une fois votre flux activé, suivez simplement les instructions présentes dans la colonne d’aide pour découvrir comment créer un compte marchand sur leDénicheur.
