Même si le critère de performance est devenu une priorité pour nos équipes depuis de nombreuses versions successives, cette nouvelle version de la solution comprend un grand nombre d’améliorations du moteur de weezbe permettant d’augmenter encore les performances de vos boutiques.

Parmi les changements majeurs, nous avons déployé un nouveau système de gestion des feuilles de styles CSS (fichiers permettant de gérer l’apparence des pages). Celles-ci sont désormais combinées, compressées et mises en cache.

Réduisant ainsi significativement le temps de chargement de toutes les pages de la boutique.

D’autre part, ce critère de performance influant directement sur le référencement (SEO) de vos boutiques, cette nouvelle version de weezbe aura également un impact positif sur ce dernier. Elle comprend d’ailleurs d’autres optimisations relatives au référencement naturel.

N’hésitez pas à consulter la liste complète des changements amenés par cette version directement sur la page d’accueil de votre interface d’administration.

Bonne ventes !
