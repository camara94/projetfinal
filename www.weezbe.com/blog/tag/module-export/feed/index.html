Nous avons le plaisir de vous annoncer la mise en production d’un nouveau module disponible dans l’espace weeStore de votre interface d’administration : l’Export vers Mondial Relay.

Vous utilisez le mode de livraison Mondial Relay sur weezbe ?

Ce module va vous permettre d’importer vos commandes prêtes à être expédiées directement dans votre compte Mondial Relay !

Mondial Relay est une solution logistique spécialisée dans la livraison de colis au particulier. Pour proposer Mondial Relay sur weezbe, rien de plus simple, il vous suffit de disposer d’un compte Mondial Relay et de configurer notre mode de livraison du même nom. Vos clients pourront alors rechercher et sélectionner le relais de leur choix directement dans le parcours d’achat de votre boutique en ligne.

Jusqu’à présent, vous deviez ensuite re-saisir les informations des commandes à expédier sur l’interface du transporteur. Terminé ! Notre module d’export sélectionne automatiquement vos commandes à expédier et vous permet de télécharger un fichier contenant toutes les informations des commandes. Il ne vous reste plus qu’à importer le fichier depuis leur interface et le tour est joué ! Vous retrouvez directement toutes les commandes pré-saisies pour l’impression des étiquettes. De quoi gagner un précieux temps et faciliter le dépôt des colis ! Le module gère le mode classique et le mode : dépôt dans un relais.

Pour bénéficier du module d’export, il suffit de vous rendre dans l’espace weeStore de votre interface d’administration weezbe et d’activer le module. Depuis cet espace, vous trouverez de nombreux modules et notamment des modules d’export vers d’autres transporteurs et solutions logistiques comme par exemple Expeditor INet pour La Poste.

N’hésitez pas à nous faire part de votre expérience sur ce module, est-ce qu’il vous facilite la vie ? Appréciez-vous travailler avec ce transporteur ? À quoi avez-vous consacré tout ce temps libre ?

Nous avons le plaisir de vous annoncer la mise en production d’un nouveau module disponible dans l’espace weeStore de votre interface d’administration : l’Export vers Expeditor INet.

Ce module va vous permettre d’exporter vos commandes prêtes à être expédiées vers le logiciel Expeditor INet de La Poste et vers le nouveau service ColiShip de Colissimo.

Expeditor I-Net est un logiciel gratuit de La Poste permettant de gérer l’envoi des colis : l’impression des étiquettes, des bordereaux de remise ou encore des CN23 pour les envois vers l’Outre-Mer et l’International.

Import des commandes dans le logiciel Expeditor INet

Expeditor INet est la solution conçue pour faciliter la gestion de vos envois So Colissimo et de tous vos envois de la gamme Colissimo Entreprise. Vous allez pouvoir optimiser vos livraisons et gagner un temps précieux. Fini d’encoder manuellement vos étiquettes, les adresses de livraison, les information du client…. notre module d’export Expeditor INet le fait pour vous ! En clic, vous téléchargez un fichier d’export contenant automatiquement les dernières commandes à expédier. Il vous suffit alors d’importer directement ce fichier dans Expeditor INet pour retrouver directement toutes les informations des commandes ! Il ne vous reste plus qu’à imprimer vos étiquettes et vos bordereaux et le tour est joué !

Si vous êtes un utilisateur de ColiShip de la plateforme Colissimo, le module génère un export compatible avec le fichier FMT standard de ColiShip permettant ainsi de migrer sur ColiShip en toute simplicité sans changer de format d’export.
Ainsi, vous pouvez continuer à importer vos commandes dans ColiShip tout comme vous le faisiez avec Expeditor INet.

Comment se procurer le module ? Tous simplement via l’interface d’administration de weezbe, dans l’espace weeStore qui regroupe tous nos modules. Vous remarquerez que de nombreux modules sont disponibles et notamment des modules d’export vers d’autres transporteurs et solutions logistiques comme par exemple Mondial Relay.
