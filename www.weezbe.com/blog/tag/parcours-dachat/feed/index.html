Ceux-ci seront affichés lors du choix d’un mode de livraison dans le parcours d’achat :

Pour définir le logo d’un mode de livraison, rendez-vous dans la rubrique « Configuration > Livraisons » et éditez le mode de livraison désiré.

Dans la colonne de gauche, cliquez sur l’icône « + » de l’encart « Logo » :

Sélectionnez alors une image par défaut (Chronopost, Colissimo suivi, ICI Relais, La poste, Mondial Relay, ou So Colissimo) ou bien importez votre propre image via le bouton « Téléchargement » :

Il ne reste plus qu’a sauvegarder votre mode de livraison !

L’ajout de logos sur les modes de livraison permet aux internautes de repérer d’un coup d’œil les possibilités de livraison qu’offre votre boutique. C’est une recommandation web-marketing supplémentaire à ne pas négliger !
