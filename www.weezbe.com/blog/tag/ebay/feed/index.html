ebay – weezbe https://www.weezbe.com Fri, 12 Oct 2018 07:32:38 +0000 fr-FR hourly 1 https://wordpress.org/?v=4.9.8 100772978 eBay est de retour sur weezbe ! https://www.weezbe.com/blog/ebay-retour-weezbe/ https://www.weezbe.com/blog/ebay-retour-weezbe/#respond Thu, 22 May 2014 09:46:24 +0000 http://www.weezbe.com/?p=4871 Continuer la lecture →]]> Vous l’avez sûrement remarqué depuis la version 1.81 de weezbe : le module de publication vers eBay est de nouveau disponible !

Disponible gratuitement (hors frais éventuels eBay), le module permet :

  • De publier vos annonces sur eBay ou sur votre boutique eBay
  • De centraliser vos commandes au sein de weezbe
  • De synchroniser vos stocks
  • D’importer vos produits sur weezbe depuis un compte eBay existant

Comment vendre sur eBay ?

Afin d’être en mesure de vendre sur eBay depuis votre boutique weezbe nous vous recommandons de prendre connaissance des liens ci-dessous :

  • Comment vendre sur eBay ?
  • Créer un compte vendeur sur eBay.fr
  • La vérification du statut professionnel sur eBay

Une fois ces étapes réalisées, rendez-vous dans la rubrique « Configuration > eBay » de votre interface d’administration afin de connecter votre compte.

Votre compte est connecté ? Vendez sur eBay et augmentez votre visibilité ainsi que votre chiffre d’affaire !

Enfin, n’hésitez pas à contacter le support technique via la rubrique « Aide » de votre interface d’administration si vous rencontrez des difficultés technique ou si vous avez des questions concernant le fonctionnement du module eBay.
