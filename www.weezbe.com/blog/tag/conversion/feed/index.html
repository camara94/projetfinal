L’objectif étant d’analyser et de suivre vos ventes au travers d’Analytics grâce à des critères tels que les transactions, les produits, le panier moyen, le taux de conversion, le délai avant l’achat, etc.

Découvrez dans cet article comment mettre en place cette fonctionnalité en quelques clics !

Google Universal Analytics

Tout d’abord, si vous ne l’avez pas encore fait, nous vous conseillons de mettre à jour votre compte Analytics vers le nouveau format nommé « Google Universal Analytics« . Pour cela, suivez simplement l’étape 1 de ce guide google.

Une fois que votre compte a été mis à jour vers Google Universal Analytics, inutile de suivre les étapes relatives au code de suivi, indiquez simplement à weezbe que vous utilisez désormais ce nouveau format depuis la rubrique « Statistiques > Webanalytics » :

Activer les rapports sur le commerce électronique

La dernière étape consiste à activer ces nouveaux rapports directement sur votre compte Google Analytics.

Dans la section « Admin » de votre compte Analytics, cliquez tout d’abord sur « Paramètres de commerce électronique » dans la colonne de droite :

Cliquez alors sur les curseurs permettant d’activer les fonctionnalités de commerce électronique et n’oubliez pas d’enregistrer :

C’est tout !

Dès que de nouvelles ventes seront réalisées via votre boutique weezbe, celles-ci seront automatiquement remontées dans la vue « Commerce électronique > Vue d’ensemble » de vos rapports statistiques :
