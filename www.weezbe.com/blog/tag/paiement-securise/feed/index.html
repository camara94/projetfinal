La solution de paiement Sips de Worldline est désormais disponible gratuitement sur weezbe !

Ce mode de paiement sécurisé vous permet de bénéficier d’une page de paiement adaptée aux supports web, tablette et mobile.Outre le paiement par carte bancaire, cette solution propose également des fonctionnalités supplémentaires telles qu’un outil de gestion de caisse, un outil de reporting ou encore d’aide à la lutte contre la fraude.

Le point fort de cette solution est la possibilité de proposer plus de 50 modes de paiements supplémentaires (Virements, PayPal, PayLib, MasterPass, 1euro.com…) en plus du paiement par carte bancaire classique. Vos clients peuvent ainsi choisir leur mode de paiement préféré sur une seule et même page de paiement. De plus, vous avez la possibilité d’opter pour une sécurité renforcée grâce à 3D Secure.

Comment installer Sips Worldline ?

Si vous possédez un compte Sips et que vous souhaitez paramétrer ce mode de paiement sur votre boutique weezbe, il vous suffit de vous rendre dans la rubrique « Configuration > Paiements » de votre interface d’administration puis de renseigner vos informations dans le module Sips Worldline.

Attention, Sips est une solution que vous devez relier à une banque ou un acquéreur, vous devez donc disposer d’un contrat de vente à distance (VAD) auprès d’une banque partenaire de Worldline.

En cas de doute, n’hésitez pas à laisser un commentaire ou contacter le support technique depuis votre interface weezbe !
