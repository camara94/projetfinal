Je vous préviens, vous allez être pris en « Flag » !

Dernière mise à jour le 11/09/2018

Aujourd’hui, je vous présente une petite nouveauté weezbe sympathique. A la fois, cette nouveauté donnera plus de style à votre boutique et améliorera le merchandising de votre espace. Je vous conseille d’activer rapidement les « flags » nouveauté et promotions pour embellir la présentation de vos produits.

En quoi ça améliore le merchandising ?

Lorsqu’un visiteur arrive dans l’un de vos rayons, il identifie facilement quelles sont les nouveautés et quelles sont les promotions dans l’ensemble des produits.

Par exemple, un promophile visualisera en 1 coup d’œil les produits qui l’intéressent : les promotions. Vous lui mettez sous le nez, ce qui le tente !

Comment activer les « flags » ou « rubans » sur ma boutique ?

Pour activer vos « flags » ou « rubans » sur votre boutique, rendez-vous dans la rubrique « Produits > affichage » de votre espace d’administration. Si vous avez créé votre boutique récemment, vous n’avez rien à faire, cette option est activée par défaut !

Comment personnaliser la couleur de mes rubans ?

Par défaut, le ruban promo est rouge et le ruban nouveauté est vert.

Je vous livre, exclusivement sur le blog, une petite astuce pour en personnaliser la couleur. Accédez à la rubrique « Configuration > CSS » et insérez le code suivant:

Transformer mon ruban promotion en orange :

.ribbons .promo {background-color:orange;}

 

Transformer mon ruban nouveauté en bleu :

.ribbons .new {background-color:blue;}

 

Vous pouvez remplacer le orange et le bleu par la couleur que vous souhaitez. Pour cela, utilisez le code hexadécimal de la couleur désirée.

J’attends vos exemples d’intégration !

Je serai très curieux de voir ce que donne les rubans avec d’autres couleurs. J’attends le lien de vos boutiques dans les commentaires pour voir les différentes intégrations sympathiques que vous avez pu mettre en place.
